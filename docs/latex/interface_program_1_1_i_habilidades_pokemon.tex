\doxysubsection{Detailed Description}
Define las propiedades básicas relacionadas con las habilidades de un Pokémon. 

$<$interface$>$ \doxylink{interface_program_1_1_i_habilidades_pokemon}{IHabilidades\+Pokemon} proporciona un contrato para implementar las características básicas de las habilidades de un Pokémon. $<$/interface$>$ 

Esta interfaz implementa el principio de responsabilidad única (SRP) del SOLID, enfocándose exclusivamente en las propiedades que definen las habilidades de un Pokémon.
